# Subscription Tracker

A retro-styled web application for managing your subscription services

# Link 
https://hwi0201.github.io/Subscription-Tracker/

## Features

- Track active and inactive subscriptions
- Monthly spending statistics and analytics
- Calendar view with payment reminders
- Notification system for upcoming payments
- Desktop interface with draggable windows

## Tech Stack

- HTML, CSS, JavaScript
- LocalStorage for data persistence
- Lucide icons

## File Structure

```
├── index.html          # Main HTML structure
├── style.css           # Styling and Y2K theme
├── app.js              # Core application logic
└── storage.js          # Data management
```

## Usage

Click the desktop icons to open different views:
- Dashboard: Overview of your subscriptions
- Calendar: Monthly payment schedule
- Notifications: Upcoming payment alerts
